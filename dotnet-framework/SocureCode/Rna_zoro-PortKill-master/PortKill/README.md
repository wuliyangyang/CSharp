# PortKlii端口占用进程杀手

![portkill](/image/portkill.png)
![debug](/image/debug.png)

## PortKlii是什么?
在做开发的时候发现总有端口被占用，这个程序是为了解决端口占用。
可以直接运行PortKill\bin\Debug目录下的PortKill.exe来启动程序。

## PortKlii有哪些功能？

* 端口占用扫描和结束占用端口的进程
* 直接输入端口号可查找并结束占用进程
* 直接执行cmd命令的调试界面
